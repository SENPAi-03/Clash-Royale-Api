# Clash-Royale-Api
**Simple Clash Royale Api Client**

## Requirements
* Api Key from https://developer.clashroyale.com
* Python Installed

## Installation
### Clone The Repo
```sh
apt-get install git
git clone https://github.com/SENPAi-03/Clash-Royale-Api
cd Clash-Royale-Api
```
### Install Requirements & Run
```sh
pip install -r requirements.txt
python main.py
```
